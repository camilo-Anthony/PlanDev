# Project Estimation Templates

This directory contains estimation templates for different project types. These templates provide realistic hour estimates based on project complexity and scope.

## Template Structure

Each template is a Markdown file that defines:
- Base hours for the project type
- Module breakdown with hour estimates
- Task-level details
- Complexity factors

## Available Templates

- `portfolio-simple.md` - Simple portfolio websites (20-40h)
- `portfolio-complex.md` - Complex portfolios with CMS (80-120h)
- `ecommerce-basic.md` - Basic e-commerce sites (150-200h)
- `saas-mvp.md` - SaaS MVP applications (300-500h)

## Usage

The AI estimation service reads these templates to generate realistic project plans based on user requirements.
